Project Overview (Digital Literacy Portfolio)

This project is designed to develop essential digital skills by making you act as a Student Digital Ambassador. Your role is to guide others in using digital tools effectively, safely, and professionally. The project consists of five practical tasks, each linked to a module, and all work is organized in a GitHub repository along with a final report.

Task-wise Overview
Task 1: Digital Literacy Infographic

You create a visual infographic using tools like Canva or Prezi.
It explains digital literacy and related topics such as tools for students, safe internet use, or email etiquette.
 Focus: Creativity + awareness building.

Task 2: Student Digital Portfolio

You build your professional online presence by creating or updating profiles on platforms like GitHub and LinkedIn.
 Focus: Career readiness + personal branding.

Task 3: Coding & Collaboration Platforms

You explore:

A coding platform (solve one beginner problem)
A Google Form quiz for digital literacy
 Focus: Practical tools for learning and collaboration.
Task 4: Email Etiquette & Online Behavior

You write professional emails and create a checklist for social media do’s and don’ts.
 Focus: Proper digital communication skills.

Task 5: Cybercrime Awareness

You prepare a case study on cybercrime (like phishing or UPI fraud) and create a prevention checklist.
 Focus: Online safety and awareness.

Final Goal

All tasks together help you:

Become digitally aware and responsible
Build a professional online identity
Learn real-world tools used in academics and careers
